# 宇宙学中的深度学习方法：章节案例代码

本仓库整理《宇宙学中的深度学习方法》中各章末尾实验案例的配套代码。案例按照“数据结构决定模型结构”的思路展开，从低维宇宙学摘要量、三维强度映射数据、引力波时间序列，到概率后验估计，展示不同深度学习模型在宇宙学问题中的典型用法。

## 案例列表

| 章节 | 建议目录 | 案例主题 | 任务类型 | 主要方法 |
|---|---|---|---|---|
| 第 2 章 | `01_BAO_Model_Classification_ANN.ipynb` | 基于 BAO 的模型区分 | 三分类：$\Lambda$CDM / $w$CDM / CPL | ANN / MLP + Cross Entropy |
| 第 3 章 | `02_hi21cm_unet.ipynb` | HI 21cm 前景减除 | 三维体素级信号重建 | PCA + 3D U-Net |
| 第 3 章 | `03_CMB_Regression_CNN_vs_ANN.ipynb` | CMB 功率谱参数回归 | 宇宙学参数点估计 | ResNet / CNN / ANN |
| 第 4 章 | `04_GW_Classification_RNN_vs_ANN.ipynb` | 基于基础 RNN 的引力波信号三分类 | BBH / Burst / Noise 分类 | Simple RNN + Cross Entropy |
| 第 5 章 | `05_SNe_Posterior_NLL_vs_NF.ipynb` | Ia 型超新星宇宙学参数后验估计 | 宇宙学参数后验估计 | Gaussian NLL + Conditional Normalizing Flow |

## 项目特点

- **面向教学与复现实验**：代码优先服务于理解模型结构、训练流程和宇宙学数据表示方式，而不是直接替代完整科研分析管线。
- **强调结构先验**：不同案例分别对应低维向量、三维数据立方体、时间序列和概率分布建模，帮助读者理解为什么要选择 ANN、CNN/U-Net、RNN 或归一化流。
- **兼顾物理解释与机器学习实现**：每个案例都对应一个具体宇宙学问题，并展示输入、标签、损失函数与评估指标之间的关系。
- **便于扩展**：可以在现有代码基础上替换模拟器、增加噪声模型、调整网络结构，或加入更严格的统计检验。

## 环境安装

建议使用 Python 3.9 或更高版本，并根据本机 CUDA 版本安装合适的 PyTorch。

## 快速开始

进入对应章节目录后运行notebook即可。

## 注意事项

这些代码主要用于配合教材理解深度学习方法在宇宙学中的应用边界。真实科研分析通常需要更完整的物理建模、误差传播、系统误差检验和统计校准。因此，建议将本仓库作为教学示例、方法原型或课程实验的起点，而不是直接作为正式宇宙学参数约束结果。
